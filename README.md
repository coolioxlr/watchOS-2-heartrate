- queary from healthkit to show heartrate


##Requirements

- Xcode 7.0, iOS 9.0 SDK, watchOS 2.0 SDK
- iOS 9.0 and watchOS 2.0

##How to build

- Change the "Team" setting on [General] for each target.
- Setup HealthKit for the WatchKit extension target.
